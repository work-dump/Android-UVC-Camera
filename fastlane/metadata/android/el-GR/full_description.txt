Αυτό το πρόγραμμα χρησιμοποιείται για τον έλεγχο κάμερας USB. Ένας προσαρμοσμένος οδηγός επιτρέπει την εκπομπή βίντεο σε όλες τις συσκευές Android με διαφορετικά chipsets. Οι συσκευές Mediathek (MTK) υποστηρίζονται, αλλά και όλες οι άλλες. Προγραμματίστε να δοκιμάσετε και να παρακολουθήσετε μια φόρμα ροής βίντεο στη usb uvc κάμερα. Λειτουργεί επίσης σε συσκευές με τσιπ βιβλιοθήκης πολυμέσων (MTK). Για να χρησιμοποιήσετε μια κάμερα USB σε συσκευές Android, απαιτείται καλώδιο USB OTG. Προαιρετικά, μπορεί να χρησιμοποιηθεί ένας διανομέας USB OTG. Δοκιμάστε διασκεδαστικά. Μια πιο λεπτομερής περιγραφή είναι διαθέσιμη στην εφαρμογή στα Αγγλικά. Περιγραφή του τρόπου χρήσης της εφαρμογής κάμερας USB με προσαρμοσμένο πρόγραμμα οδήγησης: Η ανταλλαγή δεδομένων μιας φωτογραφικής μηχανής USB γίνεται μέσω μιας ισοχρονισμένης μεταφοράς δεδομένων. Για τη μεταφορά δεδομένων εργασίας στη φωτογραφική μηχανή USB, το πρόγραμμα χρειάζεται τις μεταβλητές για την αντίστοιχη κάμερα. Μόλις ανιχνευθεί η κάμερα, οι μεταβλητές της κάμερας μπορούν να διαβαστούν αυτόματα (Ρύθμιση με ρυθμίσεις UVC). Στη συνέχεια, είναι απαραίτητο να ρυθμίσετε τις μεταβλητές που διαβάζονται έτσι ώστε η μεταφορά δεδομένων στη Κάμερα Usb να είναι όσο το δυνατόν γρηγορότερη, αλλά η συσκευή Android δεν είναι υπερφορτωμένη. Για ορεκτικά, είναι καλύτερο να επιλέξετε τα παρακάτω δεδομένα: Μέγιστο μέγεθος συσκευασίας: μεσαίο μέγεθος Πακέτα ανά αίτηση: π.χ.: 4 Ενεργά Urbs: π.χ.: 4 Οι υπόλοιπες καταχωρήσεις πρέπει να επιλεγούν σύμφωνα με τις προδιαγραφές, αλλά στη συνέχεια δεν είναι τόσο σημαντικές. Αν λάβετε πλαίσια κατά τη διάρκεια της δοκιμής, μπορείτε πραγματικά να ξεκινήσετε τη ροή βίντεο. Μπορείτε επίσης να αποθηκεύσετε τις ρυθμίσεις και να τις επαναφέρετε, φυσικά, εάν κάνετε επανεκκίνηση του προγράμματος. Οποιοσδήποτε αριθμός αποθηκεύει μπορεί να δημιουργηθεί, αλλά μπορεί επίσης να αντικατασταθεί. Με το Stream φωτογραφικής μηχανής μπορείτε επίσης να τραβάτε φωτογραφίες, οι οποίες αποθηκεύονται στο φάκελο "UVC_Camera / Pictures" στην κάρτα SD. Εδώ θα πρέπει να δώσετε στην εφαρμογή τα δικαιώματα για τη μνήμη.
