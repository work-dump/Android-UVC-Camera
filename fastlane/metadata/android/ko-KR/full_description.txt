이 프로그램은 USB 카메라를 제어하는 ​​데 사용됩니다. 사용자 정의 드라이버를 사용하면 칩셋이 다른 모든 Android 장치에서 비디오를 브로드 캐스트 할 수 있습니다. MTK (Mediathek) 장치가 지원되지만 다른 장치도 지원됩니다. USB UVC 카메라에 비디오 스트림 양식을 테스트하고 시청하는 프로그램. MTK (Media Library) 칩이있는 장치에서도 작동합니다. Android 장치에서 USB 카메라를 사용하려면 USB OTG 케이블이 필요합니다. 선택적으로 USB OTG 허브를 사용할 수 있습니다. 재미있는 테스트를 해보십시오. 더 자세한 설명은 앱에서 영어로 제공됩니다. 사용자 정의 드라이버와 함께 Usb 카메라 앱을 사용하는 방법에 대한 설명 : USB 카메라의 데이터 교환은 등시성 데이터 전송을 통해 이루어집니다. USB 카메라로 작업 데이터를 전송하려면 해당 카메라의 변수가 프로그램에 필요합니다. 카메라가 감지되면 카메라 변수를 자동으로 읽을 수 있습니다 (UVC 설정으로 설정). 그런 다음 가능한 빨리 Usb 카메라로 데이터를 전송하도록 변수를 읽도록 설정해야하지만 Android 장치에는 과부하가 걸리지 않습니다. 우선 다음 데이터를 선택하는 것이 가장 좋습니다. 최대 패키지 크기 : 중간 크기 요청 당 패킷 : 예 : 4 활성 Urbs : 예 :. : 4 나머지 항목은 사양에 따라 선택해야하지만 그렇게 중요하지는 않습니다. 테스트 실행 중에 프레임이 표시되면 실제로 비디오 스트림을 시작할 수 있습니다. 물론 프로그램을 다시 시작하면 설정을 저장하고 복원 할 수도 있습니다. 저장은 얼마든지 만들 수 있지만 덮어 쓸 수도 있습니다. 카메라 스트림을 사용하면 SD 카드의 "UVC_Camera / Pictures"폴더에 저장된 사진을 찍을 수도 있습니다. 여기에서 앱에 메모리에 대한 권한을 부여해야합니다.
