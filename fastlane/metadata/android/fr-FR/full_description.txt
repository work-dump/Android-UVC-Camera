Ce programme est utilisé pour contrôler une caméra USB. Un pilote personnalisé permet de diffuser la vidéo sur tous les appareils Android avec différents chipsets. Les périphériques Mediathek (MTK) sont pris en charge, mais tous les autres le sont également. Programme pour tester et regarder un formulaire de flux vidéo sur une caméra USB. Il fonctionne également sur les appareils dotés de puces de bibliothèque multimédia (MTK). Pour utiliser une caméra USB sur des appareils Android, un câble USB OTG est nécessaire. En option, un hub USB OTG peut être utilisé. Amusez-vous à tester. Une description plus détaillée est disponible dans l'application en anglais. Description de l'utilisation de l'application Caméra USB avec un pilote personnalisé: L'échange de données d'une caméra USB s'effectue via un transfert de données isochrone. Pour un transfert de données fonctionnel vers la caméra USB, le programme a besoin des variables pour la caméra correspondante. Une fois que la caméra a été détectée, les variables de la caméra peuvent être lues automatiquement (Configuration avec les paramètres UVC). Ensuite, il est nécessaire de définir les variables lues de sorte que le transfert de données vers Usb Camera soit aussi rapide que possible, sans que le périphérique Android ne soit surchargé. Pour commencer, il est préférable de choisir les données suivantes: Taille maximale du colis: taille moyenne Paquets par demande: par exemple: 4 Urbs actif: p.ex.: 4 Les entrées restantes doivent être sélectionnées en fonction des spécifications, mais ne sont alors pas si importantes. Si vous obtenez des images lors du test, vous pouvez réellement lancer le flux vidéo. Vous pouvez également enregistrer les paramètres et les restaurer, bien sûr, si vous redémarrez le programme. Vous pouvez créer autant de sauvegardes que vous le souhaitez, mais vous pouvez également les écraser. Avec Camera Stream, vous pouvez également prendre des photos, qui sont stockées dans le dossier "UVC_Camera / Pictures" de la carte SD. Ici, vous devez donner à l'application les autorisations pour la mémoire.
