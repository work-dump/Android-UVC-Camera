Este programa é usado para controlar uma câmera USB. Um driver personalizado permite que o vídeo seja transmitido em todos os dispositivos Android com chipsets diferentes. Os dispositivos Mediathek (MTK) são suportados, mas todos os outros também. Programa para testar e assistir a um formulário de fluxo de vídeo na câmera usb uvc. Também funciona em dispositivos com chips de biblioteca de mídia (MTK). Para usar uma câmera USB em dispositivos Android, é necessário um cabo USB OTG. Opcionalmente, um hub USB OTG pode ser usado. Divirta-se testando. Uma descrição mais detalhada está disponível no aplicativo em inglês. Descrição de como usar o aplicativo Usb Camera com um driver personalizado: A troca de dados de uma câmera USB ocorre através de uma transferência de dados isócrona. Para uma transferência de dados de trabalho para a câmera USB, o programa precisa das variáveis ​​para a respectiva câmera. Depois que a câmera for detectada, as variáveis ​​da câmera poderão ser lidas automaticamente (Configuração com configurações de UVC). Em seguida, é necessário definir as variáveis ​​lidas para que os dados sejam transferidos para a câmera USB o mais rápido possível, mas o dispositivo Android não está sobrecarregado. Para iniciantes, é melhor escolher os seguintes dados: Tamanho máximo do pacote: tamanho médio Pacotes por solicitação: por exemplo: 4 Urbs ativos.: Por exemplo: 4 As entradas restantes devem ser selecionadas de acordo com as especificações, mas não são tão importantes. Se você obtiver quadros durante a execução do teste, poderá realmente iniciar o fluxo de vídeo. Você também pode salvar as configurações e restaurá-las, é claro, se reiniciar o programa. Qualquer número de salvamentos pode ser criado, mas eles também podem ser substituídos. Com o Stream da câmera, você também pode tirar fotos, que são armazenadas na pasta "UVC_Camera / Pictures" no cartão SD. Aqui você deve conceder ao aplicativo as permissões para a memória.
